# Welcome to Goal-App

## Get started

#### 1. Install Dependencies
```bash
pnpm install
```

#### 2. Start the app

```bash
pnpm expo start
```


## Fresh New Project

#### 1 . Default

```bash
pnpm create expo-app
```

#### 2. minimalistic files(js)
```bash
pnpm create-expo-app --template blank
```
